# 🎯 الآن - افعل هذا فقط!

## ❗ المشكلة:
المزامنة لا تعمل على الأجهزة الأخرى

## ✅ الحل:

### خطوة 1️⃣ (2 دقيقة):
افتح هذا الرابط:
```
https://console.firebase.google.com/project/call-eygpt/database/rules
```

### خطوة 2️⃣ (1 دقيقة):
انسخ هذا الكود:
```json
{
  "rules": {
    ".read": true,
    ".write": true,
    "config": {
      ".read": true,
      ".write": true,
      "site": {
        ".read": true,
        ".write": true
      }
    }
  }
}
```

### خطوة 3️⃣ (1 دقيقة):
- احذف الكود الموجود
- الصق الكود الجديد
- اضغط **Publish** ✅

### خطوة 4️⃣ (1 دقيقة):
اضغط **OK** على التحذير

### خطوة 5️⃣ (بدون انتظار):
```bash
npm run dev
```

### خطوة 6️⃣ (الاختبار):
1. افتح الموقع على جهازين
2. غيّر إعداد من الأول
3. شُف الجهاز الثاني!
